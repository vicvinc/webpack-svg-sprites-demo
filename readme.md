#

## install

```bash
yarn install
```

## dev

```bash
yarn hot
```

## build

```bash
yarn build:[dev|production]
```

## 构建环境变量说明

1.  NODE_ENV: 当前 js 运行的环境，development 或者 production
2.  API_ENV: 当前 api 环境，development 或者 prodution 或者其他
